Waterfly III is an unofficiall app for your own instance of Firefly III, a free and open source finance manager.

This app requires a self-hosted Firefly III instance on your own server, please visit https://www.firefly-iii.org/ for more information about the software. It is only a companion app and cannot run standalone! You can try and test all features using the demo option available on the login screen.

Features:
* Dashboard with charts & budget overview
* Transaction list with filters
* Add & edit transactions with autocomplete, including attachments, split transactions & multi currency support
* Piggy Bank support (view & add money)
* Completely open source: https://github.com/dreautall/waterfly-iii
* No trackers

Contribution:
If you run into a bug or miss a feature, feel free to give feedback in the app, send us an email or open an issue on GitHub.
